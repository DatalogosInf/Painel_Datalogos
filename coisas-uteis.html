Coisas Uteis

Links

Abaixo estão listados alguns links que podem ser de bom uso para o suporte. Só clicar no titulo para acessar a pagina.


Links de Uso Fiscal

- Codigo CEST - Site para consulta do CEST que deve ser informado no NCM.

- Consulta CT-e - Site para consulta da CT-e. Para consulta completa será necessário o certificado.

- Consulta NF-e - Site para consulta da NF-e. Para consulta completa será necessário o certificado.

- De Olho no Imposto - Site onde é possivel fazer download das tabelas de CBENEF.

- Painel de Monitoramenteo CT-e - Consulta a disponibilidade dos ambientes de emissão de CT-e.

- Painel de Monitoramento NF-e - Consulta a disponibilidade dos ambientes de emissão de NF-e.

- Painel de Monitoramento NFC-e - Consulta a disponibilidade dos ambientes de emissão de NFC-e.

- Portal Siscomex - Na opção "Acesso Público" é possivel fazer a consulta de NCM, seja pelo código ou por uma descrição do nome do produto.

- Portal MDF-e - Site onde é possivel consultar MDF-e não encerrados, ou validar o XML para encontrar o erro.

- Receita - PR - Link utilizado para cadastrar CSC de NFC-e e tambem para solicitar a emissão de notas via sistema. Para estes casos temos na aba manuais o processo de como fazer as solicitações.

- Sites de Consulta NFC-e - Site onde contém todos os links de consulta de NFC-e.

- Validador XML - RS - Ambiente de validação de XML de NF-e trazendo o erro de forma clara, inclusive apresenta uma visualização do XML de forma mais nitida e de fácil procura.


Links de Uso nos Atendimentos Web

- Magento - Configuração de integração dos sites da K13. Acessa com o usuário do suporte.

- Integrador - Painel de configução das inegrações Plug Too e entre outros. Acessa com o usuário do suporte.

- Integração Woocommerce - Site de teste da Teorema para a integração Woocommerce. Para acessar o caminho administrador inserir "/admin" depois do ling, e no caso do cliente "/minhaloja"


Links Bancários

- Validador Banco do Brasil - Validador de remessas para o Banco do Brasil. Não é necessário fazer login somente selecionar o modelo e o arquivo.

- Validador Bradesco - Validador de remessas para o Banco Bradesco. Não é necessário fazer login somente selecionar o arquivo.

- Validador Caixa Econômica Federal - Validador de remessas para o Banco Caixa Econômica Federal. Não é necessário fazer login somente selecionar a aba "Validação Arquivo Remessa", o tipo de validação e o arquivo.

- Validador Sicoob - Validador de remessas para o Banco Sicoob. Será necessário informar os dados do beneficiário e o arquivo.

- Validador Sicredi - Validador de remessas para o Banco Sicredi, porém para este será necessário solicitar ao banco o acesso pois precisa de Usuário e Senha.


Links Uteis

- Chat GPT - IA que pode ser utilizado como auxilio para duvidas, ou até mesmo corrigir SQL de consulta.

- Convertio - Site onde é possivel converter arquivos de varias extensões, por exemplo pode ser utilizado para converter uma tabela XLSX em HTML.

- Figma - Utilizado para desenvolvimento de layouts

- Looker Studio - Pode ser utilizado para exibição de resultados de uma consulta em banco de dados, utilizando o vinculo com o Auditor.

- Remove - Pode ser utilizado para remover o fundo da imagem.
