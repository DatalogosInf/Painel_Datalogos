INSTALADOR

Instalador dos executaveis Teorema, nele há 4 tipos de instalação são eles:
1 - Instalação Terminal - Onde é instalado apenas os modulos e os componentes necessarios para poder acessar o sistema e gerar relatorios. Precisa apenas selecionar os modulos a serem instalados.

2 - Instalação Servidor - Instala o Servidor de Notas, Auditor

3 - Apenas Dependências - Será instalado apenas os compententes Firebird, Crystal e ODBC, sendo apenas uma opção para terminais que foram formatados e contém toda a estruturação em backup.
Obs: Dependencias E-Social e REINF ja constam no instalador, sempre com a mais recente.


Para download clique no botão abaixo. Irá abrir em nova guia o link.

BAT'S

Backup e Restore - O arquivo "Backup" como o proprio nome ja diz é utilizado para realizar um backup da base de dados, transformando o FDB em FBK. Ja o arquivo "Restore" realiza o inverso transforma o FBK em FDB.
Reiniciar Servidores - Utilizado para reiniciar as instancias java, finalizando o mesmo e depois iniciando novamente. Utilizado para o Fênix e Servidor de Notas
Manutenção de Banco de Dados - Arquivo utilizado para realizar manutenção no banco de dados, normalmente efetuado em casos de lentidão, estouro de erros no uso, entre outros.
IMPORTANTE: EFETUAR BACKUP ANTES DE EXECUTAR O ARQUIVO DE MANUTENÇÃO
